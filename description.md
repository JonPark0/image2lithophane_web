## 개요

이미지를 리소페인(Lithophane) 3D 모델로 바꿔주는 웹페이지 개발 프로젝트

## 요청사항

### 리소페인 종류 선택

웹 페이지 첫 화면에서 총 3가지 리소페인 형태 중에 하나를 생성할 수 있게 만든다.

- 평면형: 단일 이미지로 생성하는 평면형 리소페인 모델.
- 원통형: 단일 이미지로 생성하는 원통형 리소페인 모델. 곡면 부분에 이미지가 배치된다.
- n각기둥형: 다중 이미지(중복 가능)를 기둥의 옆면 부분들에 원하는 순서대로 나열할 수 있는 리소페인 모델. 옆면의 개수에 해당하는 n은 최소 3부터 최대 8까지로 정할 수 있게 한다.

### 이미지 작업

이미지 변환 작업의 경우 값이나 옵션이 수정되면 자동으로 적용되도록 한다.

이미지에 대해 다음과 같은 동작을 수행할 수 있어야 한다.

1. 이미지 불러오기

이미지를 불러오는 작업을 수행한다.

평면형과 원통형에서는 하나의 이미지만 불러오면 된다.

n각기둥형에서는 n의 개수만큼 이미지를 불러오면 된다.

2. 치수 지정

생성할 리소페인의 사이즈를 지정해야 한다.

평면형의 경우, 가로 및 세로 크기, 그리고 높이 항목에 대해서 가장 얇은 부분과 가장 두꺼운 부분을 지정할 수 있어야 한다.

원통형의 경우, 지름과 높이, 그리고 곡면 부분에 대해서 가장 얇은 부분과 가장 두꺼운 부분을 지정할 수 있어야 한다.

n각기둥형의 경우, 지름(중심에서 옆면의 중앙점까지의 거리)과 높이, 그리고 각 옆면 부분에 대해서 가장 얇은 부분과 가장 두꺼운 부분을 지정할 수 있어야 한다.

3. 지정된 치수에 맞춰 이미지 조정

- 늘리기(stretch): 원본 이미지의 가로세로 비율을 무시하고 지정한 리소페인 사이즈에 딱 맞게 늘리거나 줄이는 것.
- 맞추기(fit): 원본 이미지의 가로세로 비율을 유지하면서 지정한 리소페인 사이즈 안에 이미지가 전부 보이도록 조절하는 것. 이미지의 가로나 세로 중 더 긴 쪽을 리소페인에 딱 맞추는게 통상적임. 이미지 이외 공간에 여백이 생길 수 있음.
- 채우기(cover): 원본 이미지의 가로세로 비율을 유지하면서 지정한 리소페인 사이즈 안에 여백이 없게끔 이미지를 채우는 것. 여백은 안 생기지만 이미지가 잘릴 수 있음.
- 타일(tile): 바둑판식 배열이라고도 함. 보통 사이즈가 작은 이미지에 사용함. 지정한 리소페인 사이즈 범위 내에서 이미지를 반복하여 바독판 타일처럼 채우는 것.

### 리소페인 변환 작업

리소페인 변환 작업은 "변환하기" 버튼을 누르면 진행되게끔 만들 것.

필요한 이미지들이 전부 준비되었다면 해당 이미지들을 가지고 지정된 형태로 리소페인을 생성하도록 한다.

원통형의 경우, 원형인 부분들에 해당하는 윗면과 밑면 부분에 대해 생성 여부를 선택할 수 있게끔 할 것.

마찬가지로 n각기둥형에서도 n각형에 해당하는 윗면과 밑면 부분에 대해 생성 여부를 선택할 수 있게끔 할 것.

### 미리보기 기능

생성된 리소페인 모델을 3차원으로 볼 수 있게끔 한다.

### 파일 다운로드 기능

생성된 파일을 stl 형식으로 다운로드 할 수 있게 한다.
